# Ork Core

Core bits of the Ork toolset.

## Requirements
* PHP 7.1

## Installation

### Via command line:
```bash
$ composer require ork/core
```

### Via composer.json:
```json
"require": {
    "ork/core": "^1.0.0"
},
```

## Documentation

See the [wiki](https://github.com/AlexHowansky/ork-core/wiki).

## Development

### Testing
```bash
$ composer test
```

### Generate API Documentation
```bash
$ composer doc
```

### Code Styling Validation
```bash
$ composer phpcs
```
