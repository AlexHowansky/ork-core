# \Ork\Core

Core bits of the Ork toolset.

# Requirements
* PHP 7.1

# Installation

## Via command line:
```bash
$ composer require ork/core
```

## Via composer.json:
```json
"require": {
    "ork/core": "*"
},
```

# Generate Documentation
```bash
$ composer doc
```

# Testing
```bash
composer test
```

# Code Styling Validation
```bash
composer phpcs
```
