# Ork Core

Core bits of the Ork toolset.

## Requirements
* PHP 7.1

## Installation

### Via command line:
```bash
$ composer require ork/core
```

### Via composer.json:
```json
"require": {
    "ork/core": "*"
},
```

## Development

### Testing
```bash
$ composer test
```

### Generate Documentation
```bash
$ composer doc
```

### Code Styling Validation
```bash
$ composer phpcs
```
