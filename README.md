# Ork Core

Core bits of the Ork toolset.

## Requirements
  * PHP 7.2

## Installation

### Via command line
```bash
$ composer require ork/core
```

### Via composer.json
```json
"require": {
    "ork/core": "^1.0.0"
},
```

## Documentation

See the [wiki](https://github.com/AlexHowansky/ork-core/wiki).

## Development

### Code Style Validation
```bash
$ composer phpcs
```

### Static Analysis
```bash
$ composer phpstan
```

### Unit Testing
```bash
$ composer test
```
